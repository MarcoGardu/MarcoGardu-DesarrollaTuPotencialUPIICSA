Conoce JFLAP

JFLAP es un software educativo interactivo escrito en Java, experimenta con temas en la informática, área de lenguajes formales y teoría de autómatas.
Para ejecutar este programa solo necesitas tener java instalado en tu equipo.

Awk, simplifica tu vida

Diseñado para procesar datos basados en texto, ya sean ficheros o flujos de datos. AWK es ejemplo de un lenguaje de programación que usa ampliamente el tipo de datos de listas asociativas (es decir, listas indexadas por cadenas clave), y expresiones regulares.

Ejercita tu mente

Un ejercicio para poner en practica JFLAP, puede ayudarte para la resolucion de ejercicios en clase así como el estudio del software.
